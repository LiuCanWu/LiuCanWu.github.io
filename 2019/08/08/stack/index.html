堆栈

数学表达式中有中缀表达式与后缀表达式，如 a + b * c - d / e 和 abc *+ de /-

需要某种存储方法，能顺序存储运算数，并在需要时“倒序输出”

堆栈(Stack): 具有一定操作约束的线性表，且只在一端(栈顶 ，TOP)做插入(入栈，Push)，删除(出栈，Pop),后入先出(LIFO)的存储顺序，Push 与 Pop 可以穿插交替进行

操作

  1. 栈的顺序存储结构：通常由一个一维数组和一个记录栈顶元素位置的变量组成
